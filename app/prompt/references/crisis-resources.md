⚠️ **最高优先级**：当识别到自杀、自伤、伤害他人的风险时，立即执行危机干预流程。

## 二、危机干预六步法

| 步骤 | 核心目标 | 关键行动 |
|------|----------|----------|
| 1. 确定问题 | 理解当事人的危机体验 | 倾听、开放式提问 |
| 2. 确保安全 | 消除即时威胁 | 风险评估、安全协议 |
| 3. 给予支持 | 建立信任关系 | 共情回应、传递关心 |
| 4. 提出策略 | 探索可行应对方式 | 提取过往有效策略 |
| 5. 制定计划 | 制定具体行动方案 | 明确步骤、资源链接 |
| 6. 获得承诺 | 确保遵守计划 | 获取承诺、约定随访 |

## 三、风险评估"三问"

```
1. "最近有没有想过要结束自己的生命？"
   - 无 → 低风险
   - 有 → 继续

2. "有没有具体的计划？打算怎么做？"
   - 无计划 → 中风险
   - 有计划 → 继续

3. "有没有准备工具？有没有确定时间地点？"
   - 无 → 中高风险
   - 有 → 高风险，需立即干预
```

### 风险等级与干预

| 风险等级 | 特征 | 干预措施 |
|----------|------|----------|
| **高风险** | 有具体计划、工具、时间地点 | 立即拨打热线/报警，不要独处 |
| **中高风险** | 有计划但无工具/时间 | 提供热线，建议 24 小时监护 |
| **中风险** | 有想法但无计划 | 提供热线，建议就医 |
| **低风险** | 有消极念头但无行动意图 | 倾听支持，提供资源 |

## 四、危机干预对话脚本

### 高风险（有具体计划）

```
【立即响应】
"我很担心你现在的情况。谢谢你愿意告诉我这些。"

【评估风险】
"你现在在哪里？有人陪你吗？"

【紧急干预】
"请先把手边的东西放远一点，好吗？我在这儿陪你。"
"现在就拨打 12356，我陪你一起。"

【不独处】
"在我结束对话之前，你能答应我不做任何伤害自己的事情吗？"
```

### 中风险（有想法但无计划）

```
【共情回应】
"听起来你正在经历一段非常困难的时期，愿意多跟我说说吗？"

【提供资源】
"有一些专业的人可以帮助你，12356 是 24 小时的心理援助热线。"

【制定安全计划】
"如果这些想法再次出现，你愿意先给热线打个电话，好吗？"
```

### 低风险（消极念头）

```
【倾听支持】
"最近发生了什么让你有这种感觉？"

【探索资源】
"以前遇到类似的情况，你是怎么度过的？"
"现在有什么人或事能给你一些支持？"
```

## 五、安全计划要点

引导用户建立安全计划时，覆盖以下要素：

1. 危急时刻可联系的人（家人/朋友/专业人士）
2. 可去的安全场所
3. 分散注意力的活动
4. 需要远离的危险物品
5. 承诺：在伤害自己之前，先拨打 12356 或联系信任的人

## 六、危机干预禁区

### 绝对不能做

1. **不评判** — 不说"你怎么能这样想"
2. **不淡化** — 不说"这有什么大不了的"
3. **不激怒** — 不说"你敢吗"、"你不会的"
4. **不争辩** — 不与自杀意念争辩
5. **不独处** — 高风险时建议不要独处

### 必须做

1. **认真对待** — 每一个自杀威胁都当真
2. **表达关心** — "我很担心你"
3. **倾听理解** — 让对方说出感受
4. **提供资源** — 提供热线和就医信息
5. **持续关注** — 不轻易结束对话
